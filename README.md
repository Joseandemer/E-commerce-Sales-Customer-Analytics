📊 Análisis de Ventas y Clientes en E-commerce

SQL + Power BI | Modelo Relacional

📌 Descripción

Este proyecto analiza el desempeño de ventas y el comportamiento de clientes en un entorno de comercio electrónico, utilizando un modelo de datos relacional compuesto por clientes, pedidos y productos.

Se emplearon consultas SQL (JOINs, agregaciones y CTEs) para construir KPIs clave del negocio, y posteriormente se desarrolló un dashboard interactivo en Power BI para la visualización de insights estratégicos.

⚙️ Tecnologías
SQL (MySQL)
Power BI
Excel
📊 KPIs principales
Clientes totales
Volumen de productos vendidos
Ingresos totales
Ticket promedio
Ingresos por ciudad
Top productos y clientes
Tendencia de ventas
🧠 Insights clave
Bogotá lidera la generación de ingresos
El año 2023 presenta el mayor pico de ventas
Existe concentración de ingresos en clientes de alto valor
La categoría Electronics es el principal motor del negocio
📈 Resultado

Se construyó un dashboard interactivo que permite analizar ventas, clientes y productos, facilitando la toma de decisiones basada en datos.
